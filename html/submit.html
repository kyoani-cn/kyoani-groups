如何提交群信息

通过GitHub提交

  1. 访问项目的GitHub仓库
  2. 找到 groups.json 文件
  3. 点击编辑按钮（铅笔图标）
  4. 按照现有格式添加您的群组信息
  5. 提交Pull Request等待审核

通过维护人提交

如果您不熟悉GitHub操作，可以通过以下方式联系维护人：

  • QQ：
  • 邮箱：

信息格式说明

{
    "name": "群组名称",
    "number": "群号",
    "region": "地区（国家/省份/都道府县）",
    "city": "城市",
    "contact": "负责人QQ"
}
